# music-transfer-helper-user-guide
переносит треки из вк, яндекса и прочего в спотифай

что делает скрипт:
копирует название трека и исполнителя из твоего плейлиста.

сохраняет всё это чудо юдо в txt файл.

сортирует треки и делает их красиво читаемыми в формате: "название трека — исполнитель"

сохраняет результат в папку list с именем имя_файла_list.txt


как пользоваться:

скопируй плейлист в txt, где каждая песня и исполнитель на отдельной строке (трек, потом артист).

запусти скрипт format_songs.py.

введи имя файла без .txt и нажми Enter.

скрипт создаст готовый файл list/имя_файла_list.txt.

идём на ресурс https://www.spotlistr.com/search/textbox, логинимся в спотифай.

вставляем готовый текст из _list.txt в нужное поле.

вводим название плейлиста и ловим кайф.


советы:

можно обрабатывать сколько угодно исходных файлов подряд — скрипт спрашивает имя каждый раз.

чтобы выйти из скрипта — введи exit.
